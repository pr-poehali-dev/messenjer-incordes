## Color Palette Reference

### Backgrounds
- `#2B2D31` - Main background
- `#313338` - Chat area, cards
- `#1E1F22` - Server sidebar, inputs
- `#232428` - User bar
- `#35373C` - Hover states
- `#404249` - Active/selected states
- `#383A40` - Input fields

### Text
- `#FFFFFF` - Primary text
- `#B5BAC1` - Secondary text
- `#6D6F78` - Muted text/placeholders
- `#DBDEE1` - Message content

### Accents
- `#5865F2` - Primary accent (Incordes blue)
- `#4752C4` - Primary accent hover
- `#3BA55D` - Success green
- `#F23F42` - Error red
- `#00A8FC` - Link blue

## Responsive Breakpoints

```
Mobile:     < 768px   - Stack vertically, hide server sidebar
Tablet:     768-1024px - Show server + channels or chat
Desktop:    > 1024px  - Show all three columns (full layout)
```

## Animations

- **Hover transitions:** 200ms ease
- **Route transitions:** Fade in/out
- **Modal open/close:** Scale + fade
- **Button clicks:** Scale down slightly
- **Server icon hover:** Rounded corners (50% â†’ 30% border-radius)
- **Scroll:** Smooth scroll behavior

## Accessibility Features

- **Focus indicators:** Blue ring on keyboard focus
- **ARIA labels:** All interactive elements
- **Keyboard navigation:** Tab order, Enter to submit
- **Screen readers:** Semantic HTML, role attributes
- **Color contrast:** WCAG AA compliant
- **Alt text:** All images and avatars
